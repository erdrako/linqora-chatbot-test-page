Generated from `erdrako/linqora-project`.

`linqora-project/docs/governance/branch-promotion-policy.md`.

Operational notes:

- `main` is production for repos with a production branch.
- `development` is the integration branch when present.
- Reconciliation between `main` and `development` must happen through reviewable
  pull requests.
- No force push or destructive history rewrite is allowed.
